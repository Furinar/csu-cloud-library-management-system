# 中南云图书管理系统

## 1 技术栈

### 前端

- **核心框架**: Vue 3 + TypeScript
- **构建工具**: Vite 5
- **UI 组件库**: Element Plus
- **路由管理**: Vue Router 4
- **状态管理**: Pinia
- **HTTP 客户端**: Axios

### 后端

- **核心框架**: Spring Boot 3.5.7
- **ORM 框架**: MyBatis 3.5.7
- **工具库**: Lombok, Hutool
- **数据库**: MySQL / MariaDB

### 部署与运维

- **构建工具**: Maven 3
- **接口测试**: apifox
- **版本控制**: Git

## 2 项目结构

```
├── 用户认证与权限模块
│   ├── 用户登录（含身份验证）
│   ├── 注销登录
│   ├── 密码重置

├── 图书资源管理模块
│   ├── 新书推荐（最新上架TOP5展示）
│   ├── 图书查询（条件查询、分页展示、检索）
│   ├── 新增图书（管理员权限）
│   ├── 编辑图书（管理员权限，仅可借阅状态）
│   ├── 图书编目与库存管理
│   └── 图书基础配置（分类、状态等）
├── 借阅流程管理模块
│   ├── 图书借阅（含归还时间校验）
│   ├── 归还申请（用户发起）
│   ├── 归还确认（管理员权限）
│   ├── 续借
│   ├── 逾期处理
│   └── 借阅记录生成（归还确认后自动记录）
├── 系统管理模块
│   ├── 用户管理（管理员维护用户信息）
│   ├── 操作日志记录
│   ├── 系统消息推送
│   └── 系统基础设置
└── 数据统计与报表模块
    ├── 统计仪表盘（可视化展示）
    ├── 图书统计（总藏书量、分类统计、热门TOP10）
    ├── 用户统计（注册用户数、活跃用户数、借阅排行）
    ├── 借阅统计（月度/年度借阅量、逾期率、归还率）
    ├── 明细统计查询
    └── 报表生成与导出（Excel/PDF）
```

## 3 项目使用指南

### 环境要求

- **Node.js**: >= 18.0.0
  下载网址:https://nodejs.org/zh-cn/
- **包管理器**: npm (>= 9.0.0)

### 安装流程

1. **获取代码**

   ```bash
   git clone https://github.com/Furinar/csu-cloud-library-management-system.git
   cd csu-cloud-library-management-system
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

### 运行开发环境

```bash
npm run dev
```

启动成功后，控制台将显示本地访问地址（通常为 `http://localhost:5173`）。

## 4 功能详细设计

### 一、用户认证与权限模块

#### 1. 用户登录（含身份验证,包含用户端和管理员端）

- **功能概述**: 验证用户身份，授予系统访问权限
- **操作流程**:
  1. 用户输入账号（手机号/用户名/邮箱）+ 密码
  2. 系统验证验证码（图形/短信，可选）
  3. 系统校验账号密码正确性
  4. 验证通过后创建会话，记录登录状态
- **权限要求**: 无（公开功能）
- **异常处理**: 账号不存在/密码错误提示；账号冻结提示；连续输错锁定。

#### 2. 注销登录

- **功能概述**: 销毁当前用户会话，退出系统
- **操作流程**: 点击“注销” -> 清除 Token -> 跳转登录页。

#### 3. 密码重置

- **功能概述**: 帮助用户找回或修改密码（忘记密码/已登录改密）。

#### 4. 角色权限控制

- **角色定义**:
  - **管理员**: 拥有所有功能权限。
  - **普通用户**: 个人中心、图书查询、借阅归还操作。
- **控制方式**: RBAC（角色-权限-资源）模型。

### 二、个人中心模块

#### 1. 个人信息管理

- **功能概述**: 查看/修改个人基础信息（姓名、性别、联系方式等）。

#### 2. 我的借阅

- **功能概述**: 展示当前借阅且未归还的图书。
- **操作**: 查看详情、申请续借、查看逾期提醒。

#### 3. 借阅历史

- **功能概述**: 展示已归还的借阅记录，支持按时间/状态查询及导出。

#### 4. 消息通知查看

- **功能概述**: 接收系统公告、借阅提醒、个人通知。

### 三、图书资源管理模块

#### 1. 新书推荐

- **规则**: 按上架时间倒序取 TOP5。
- **展示**: 封面、书名、作者、简介。

#### 2. 图书查询

- **条件**: 书名、作者、ISBN、分类、年份、状态。
- **结果**: 分页展示，支持排序。

#### 3. 新增图书 (管理员)

- **录入**: 基础信息、分类、库存、封面等。
- **验证**: ISBN 唯一性。

#### 4. 编辑图书 (管理员)

- **范围**: 简介、分类、位置、状态、增加库存。
- **限制**: 有借出记录时不可修改核心标识。

#### 5. 图书编目与库存管理

- **编目**: 维护分类体系。
- **库存**: 查看明细、库存变更、批量导入。

#### 6. 图书基础配置

- **配置**: 分类管理、状态定义、借阅规则设置。

### 四、借阅流程管理模块

#### 1. 图书借阅

- **流程**: 校验条件（状态、限额、逾期） -> 生成记录 -> 更新状态 -> 计算应还日期。

#### 2. 归还申请 (用户)

- **流程**: 选择图书 -> 填写说明 -> 提交申请。

#### 3. 归还确认 (管理员)

- **流程**: 核对实物 -> 确认归还 -> 更新状态 -> 处理逾期。

#### 4. 续借

- **续借**: 未逾期/未超限可续借。

#### 5. 逾期处理

- **机制**: 多级提醒（1 天/3 天/7 天），逾期限制权限，支持手动处理。

#### 6. 借阅记录生成

- **机制**: 归还确认后自动归档，支持统计查询。

### 五、系统管理模块

#### 1. 用户管理

- **功能**: 用户列表、增删改查、禁用/启用、重置密码。

#### 2. 操作日志记录

- **内容**: 记录关键操作（时间、人员、内容、结果），支持审计。

#### 3. 系统消息推送

- **功能**: 发送全员或定向消息（公告/提醒）。

#### 4. 系统基础设置

- **配置**: 系统信息、借阅规则参数、安全设置。

### 六、数据统计与报表模块

#### 1. 统计仪表盘

- **展示**: 核心指标、趋势图、占比图、TOP 排行。

#### 2. 图书统计

- **维度**: 总藏书量、分类占比、热门图书 TOP10。

#### 3. 用户统计

- **维度**: 注册/活跃用户、借阅排行、用户画像。

#### 4. 借阅统计

- **维度**: 借阅量趋势、逾期率、归还率、高峰分析。

#### 5. 明细统计查询

- **功能**: 多维度钻取（图书/用户/时间），支持导出。

#### 6. 报表生成与导出

- **格式**: Excel (数据), PDF (图表)。

## 5 美学设计

1.  **主色调规范**:
    - **基底色**: 纯净白色 (RGB 255,255,255)，占比 75%-80%。
    - **核心色**: 核心蓝色 (RGB 22,92,145)，占比 15%-20%，用于关键交互、标题、焦点。
    - **辅助色**: 低饱和浅灰 (RGB 240,242,245)，用于次要区域。
2.  **风格基调**: 极简主义，大气高端。强调留白、利落线条、清晰层次。轻微阴影 (<2px) 和极简渐变。
3.  **核心界面视觉**:
    - **登录页**: 左右分栏或居中卡片，白色底，蓝色装饰，可加低透明度书页纹理。
    - **书架页**: 卡片式布局，hover 时蓝色边框，边缘 1px 细腻描边。
    - **检索页**: 突出搜索框，聚焦变蓝，结果选中高亮淡蓝。
4.  **元素设计规范**:
    - **图标**: 线性风格，2px 粗细，常态深灰，激活核心蓝。
    - **按钮**: 圆角矩形 (4px)，主按钮蓝底白字，次按钮白底蓝边。
    - **字体**: 无衬线字体 (思源黑体)，标题核心蓝 (600)，正文深灰 (400)，辅助中灰。
5.  **细节质感**: 微动画 (0.3s 缩放/淡入)，玻璃拟态弹窗 (5px 模糊)，精细图表配色。
6.  **整体氛围**: 兼顾“文化属性”与“科技感”，使用真实 UI 图片及中南大学校徽元素。

## 6 交互设计

1.  **核心原则**: 简洁高效、逻辑连贯、反馈明确。路径最短化。
2.  **关键流程交互**:
    - **检索**: 实时联想，多维排序，筛选反馈。
    - **借阅**: 极简确认弹窗，渐变成功提示，状态实时更新。
    - **管理**: 批量操作支持，列表无感知刷新。
3.  **界面元素交互**:
    - **导航**: 下划线标识，hover 变色，下拉淡入。
    - **卡片**: hover 上浮+高亮，点击无延迟，收藏微交互。
    - **表单**: 聚焦变色，错误红字提示，下拉项 hover 淡蓝。
4.  **反馈设计**:
    - **操作**: 成功(蓝)/警告(橙)/错误(红) 统一图标文字提示。
    - **加载**: 极简蓝色环形动画，列表滚动加载反馈。
    - **滚动**: 导航固定，回到顶部按钮，平滑滚动。
5.  **适配与易用性**:
    - **跨端**: PC 拖拽/移动端触摸，触觉反馈。
    - **权限**: 按钮权限控制 (置灰+提示)。
    - **辅助**: 快捷键支持 (Ctrl+F, Ctrl+S)。
